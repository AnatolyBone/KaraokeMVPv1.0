# 🚀 ДОРОЖНАЯ КАРТА РАЗВИТИЯ ПРОЕКТА: “Karaoke LRC Maker”

В данном документе описаны будущие приоритетные шаги по совершенствованию веб-приложения для создания караоке-разметки.

---

## 📅 Этап 1: Расширение возможностей синхронизации (Приоритет 1)
- [x] **Интеграция с базой текстов LRCLIB и автопоиск:** Парсинг ID3-тегов из аудио для автоматического формирования поискового запроса, быстрый поиск и импорт текстов с таймингами (LRC) или без них (Plain) через единое модальное окно.
- [ ] **Посимвольная синхронизация (Character-by-character sync):** Внедрение посимвольной разметки для языков со сложной слоговой структурой (например, азиатские языки).
- [ ] **Waveform Zooming:** Возможность масштабировать колесиком мыши интерактивную аудиоволну для сверхточной расстановки меток на стыке слов.
- [ ] **Автоматическое выравнивание по сетке:** Подсказка оптимальных таймингов на основе спектрограммы голоса (выделение вокального диапазона).

---

## 🎨 Этап 2: Дизайн, Шаблоны и Кастомизация видео (Приоритет 2)
- [x] **Видео-фоны (Dynamic Video Backgrounds):** Поддержка загрузки пользовательских фоновых видеофайлов (.mp4) и запуск генеративных эффектов (жидкий градиент `fluid-gradient`, снегопад `snow`, пыльца `lens-dust`).
- [x] **Анимации караоке (Kara-style presets):** Разработка нескольких пресетов анимации текста:
  * *Apple Music Style:* Текст плавно увеличивается и размывается при смене.
  * *Classic Karaoke:* Заливка текста слева направо сплошным цветом.
  * *Split-Screen Style:* Разделение экрана с обложкой наверху/слева и списком строк, скроллящихся с эффектом фокусировки.
- [ ] **Кастомные шрифты по URL:** Возможность загрузить любой TTF/OTF шрифт через Google Fonts для уникального дизайна титров.

---

## 🤖 Этап 3: Автоматизация и искусственный интеллект (Приоритет 3)
- [ ] **Интеграция Whisper API:** Автоматическое распознавание речи из загруженного аудиофайла и автоматическая разбивка текста по строкам.
- [ ] **WhisperX alignment:** Автоматическая расстановка пословных таймингов (AI-ассистент) на основе спектрального анализа аудио, исключая необходимость ручного простукивания пробелом!
- [ ] **AI-переводчик:** Автоматический перевод песен на любой язык одной кнопкой с интеграцией Deepl/GPT для мультиязычных субтитров.

---

## 🔗 Этап 4: Sharing и Облачные интеграции (Приоритет 4)
- [ ] **Облачное сохранение проектов:** Возможность сохранять свои проекты в облачную базу (Firebase / Supabase) для доступа с любого устройства без IndexedDB.
- [ ] **Прямой шеринг в соцсети:** Генерация коротких видео-превью караоке (клипов до 30 секунд) для быстрой публикации в YouTube Shorts, TikTok и Instagram Reels.
- [ ] **Совместный режим (Real-time Collaboration):** Совместная расстановка таймингов несколькими пользователями одновременно через WebSocket-соединение.
