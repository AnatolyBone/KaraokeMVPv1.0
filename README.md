# 🎤 Karaoke LRC Maker

Веб-приложение нового поколения для создания профессиональной караоке-разметки песен, синхронизированных субтитров (LRC) и видеороликов прямо в вашем браузере!

Вся обработка медиафайлов, декодирование и рендеринг происходят **локально на стороне клиента**, без отправки аудио или текстов на внешние сервера, гарантируя абсолютную конфиденциальность и высокую скорость работы.

---

## ✨ Ключевые особенности и возможности

1. **Двухрежимное рабочее пространство (Workspace):**
   * **Режим Записи (Timing Sync):** Чистый фокус на расстановке временных меток под музыку. Поддерживает три режима синхронизации:
     * *Построчно (Line Sync):* Проставление таймингов для всей строки целиком.
     * *Пословно (Word Sync):* Разметка каждого слова отдельно для плавного пословного караоке-закрашивания.
     * *Послогово (Syllable Sync):* Высокоточная разметка слогов. Слоги генерируются автоматически с помощью встроенного алгоритма слогораздела ([hyphenation.ts](file:///Users/anatoly/Yandex.Disk.localized/my%20projects/karaoke/karaoke-lrc-maker-mvp/src/utils/hyphenation.ts)) для русского и английского языков.
     * Разметка выполняется нажатием клавиши `Space` (или кликами по большой адаптивной Tap-зоне на мобильных устройствах).
   * **Режим Коррекции (Fine-Tuning):** Инструменты для редактирования и выравнивания:
     * *Визуальный таймлайн ([TimelineEditor.tsx](file:///Users/anatoly/Yandex.Disk.localized/my%20projects/karaoke/karaoke-lrc-maker-mvp/src/components/TimelineEditor.tsx)):* Горизонтальная шкала времени с перетаскиванием блоков строк. Поддерживает **Магнитную привязку (Magnetic Snapping)** к долям бита (BPM) и границам соседних строк с радиусом захвата в 220 мс и визуальным индикатором `МАГНИТ ✓`.
     * *Таблица строк ([LyricsTable.tsx](file:///Users/anatoly/Yandex.Disk.localized/my%20projects/karaoke/karaoke-lrc-maker-mvp/src/features/lyrics/LyricsTable.tsx)):* Позволяет выполнять точечный сдвиг времени (с шагом `±0.1` сек), сбрасывать тайминги, удалять строки, разделять фразы (`splitLine`) начиная с выбранного слова и объединять строки (`mergeLines`). Поддерживает редактирование текста на лету и микротайминг слов.

2. **Мультиязычность и Параллельный перевод:**
   * Возможность добавления параллельного перевода для каждой строки. Перевод отображается непосредственно под оригинальным текстом в живом плеере ([KaraokePreview.tsx](file:///Users/anatoly/Yandex.Disk.localized/my%20projects/karaoke/karaoke-lrc-maker-mvp/src/features/preview/KaraokePreview.tsx)) и накладывается на экспортируемое видео (поддерживается в стилях *Apple Music*, *Spotify*, *TikTok Neon* и *Classic*, но не отрисовывается в режиме *Split-Screen*).
   * Интерфейс полностью переведен на русский и английский языки с переключением на лету.

3. **Автоматический анализ BPM & Сетка битов:**
   * Встроенный Web Audio декодер считывает PCM-данные трека, строит интерактивную аудиоволну ([Waveform.tsx](file:///Users/anatoly/Yandex.Disk.localized/my%20projects/karaoke/karaoke-lrc-maker-mvp/src/components/Waveform.tsx)) и автоматически определяет темп (BPM) по пикам локальной звуковой энергии.
   * Волна поддерживает интерактивное масштабирование от 1x до 10x (кнопками управления или через `Ctrl + Wheel`) с автоматической прокруткой вслед за плейхедом.
   * Эмулирует частотную спектрограмму (частотный водопад) в нижней трети аудиоволны.
   * Опция **Привязки к биту (Snap to Beat)** автоматически корректирует ваши клики `Space` к ближайшей музыкальной доле (с окном захвата 250 мс).

4. **Высокопроизводительный Cinema Engine (Экспорт видео):**
   * **Режим А (Аппаратное ускорение WebCodecs):** Сверхбыстрый оффлайн-экспорт видео высокой четкости 720p/1080p (MP4 / WebM) без тормозов интерфейса. Использует нативный `VideoEncoder`/`AudioEncoder` с аппаратным ускорением (prefer-hardware) и оффлайн-мультиплексированием через библиотеки `mp4-muxer` и `webm-muxer` на частоте 30 FPS.
   * **Режим Б (Медиа-рекордер):** Автоматический реалтайм-fallback для браузеров без поддержки WebCodecs (например, Safari). Захватывает поток с canvas (`canvas.captureStream(60)`) и аудиопоток из AudioContext на частоте 60 FPS. Работает через Web Worker таймер для обхода троттлинга фоновых вкладок браузера.
   * **Послойный Профайлер:** Отображает FPS, общее время рендеринга кадра и детальный мониторинг времени отрисовки каждого слоя (Background, Particles, Lyrics Layer, Visualizer) в режиме реального времени.
   * **Оптимизации производительности:** Предварительный рендеринг и умное кэширование ширины текста ([textCache.ts](file:///Users/anatoly/Yandex.Disk.localized/my%20projects/karaoke/karaoke-lrc-maker-mvp/src/utils/renderer/textCache.ts)) исключают мерцание шрифтов. Для размытия фонов обложек (`cover-blur`) и эффекта `fluid-gradient` используется оффскрин-холст в масштабе 0.25x, что ускоряет рендеринг до 16 раз.
   * **Векторное сглаживание:** Векторное масштабирование обложек высокого разрешения с качеством (`imageSmoothingQuality = 'high'`).

5. **Премиальный дизайн и анимации:**
   * **Пресеты брендов:** Стили караоке Apple Music, Spotify, TikTok Neon, Classic Karaoke и Minimal Cinema.
   * **Живые эффекты (Canvas 2D):** Эффекты частиц (падающий снег `snow`, золотистая пыльца `lens-dust`), фоновые анимированные радиальные градиенты (`fluid-gradient`), размытие обложки или пользовательские фоновые видеоклипы (.mp4).
   * **Форматы кадра:** Поддержка горизонтального (16:9), мобильного вертикального (9:16) и квадратного (1:1) соотношения сторон.
   * В вертикальном (9:16) и квадратном (1:1) форматах плеера используется макет **Split-Screen**: обложка и название трека вынесены в красивую верхнюю плашку, список строк ограничен 5 строками с эффектом центрирования барабана, а прогресс-бар и счетчики времени (`MM:SS`) вынесены в нижнюю панель.

6. **Локальное кэширование и менеджер проектов:**
   * Автоматическое кэширование настроек и текущего текста в `localStorage` (через Zustand `persist`).
   * Сохранение аудиофайлов и обложек в **IndexedDB** ([db.ts](file:///Users/anatoly/Yandex.Disk.localized/my%20projects/karaoke/karaoke-lrc-maker-mvp/src/utils/db.ts)) для предотвращения сброса при перезагрузке страницы.
   * **Менеджер проектов ([RecentProjects.tsx](file:///Users/anatoly/Yandex.Disk.localized/my%20projects/karaoke/karaoke-lrc-maker-mvp/src/components/RecentProjects.tsx)):** Возможность сохранять, загружать и удалять недавние проекты локально.
   * Поддержка глобальной **Истории изменений (Undo/Redo)** на уровне Zustand store.

7. **Интеграция с LRCLIB и нативный парсер ID3:**
   * **Нативный ID3-парсер ([metadata.ts](file:///Users/anatoly/Yandex.Disk.localized/my%20projects/karaoke/karaoke-lrc-maker-mvp/src/utils/metadata.ts)):** При загрузке аудиофайла клиентская часть считывает теги ID3v2 (v2.2, v2.3, v2.4) для Artist, Title и Album. Поддерживает декодирование различных кодировок строк (UTF-8, UTF-16, ISO-8859-1) и исправляет битые кодировки кириллицы (Windows-1251, ошибочно определенные как Latin1).
   * **Парсер обложек ([cover.ts](file:///Users/anatoly/Yandex.Disk.localized/my%20projects/karaoke/karaoke-lrc-maker-mvp/src/utils/cover.ts)):** Читает бинарные APIC/PIC фреймы или находит картинки в файле по сигнатурам (JPEG/PNG magic bytes).
   * **LRCLIB Поиск и автоимпорт:** Поиск по базе текстов и LRC-файлов API `https://lrclib.net` по кнопке или автоматически при загрузке аудио. Запросы защищены таймаутом `AbortController`. При сбое прямого запроса используется резервный прокси-сервер CORS (`allorigins`).
   * **Импорт и Экспорт файлов:** Поддержка караоке LRC (включая Enhanced LRC с пословными тегами `<>`), SubRip (SRT), WebVTT (VTT) и SubStation Alpha (ASS). Экспорт в ASS поддерживает караоке-теги `{\k}` в сантисекундах.

*Примечание: Вспомогательные утилиты ZIP-архивации проекта и генерации iframe-кодов в файле [sharing.ts](file:///Users/anatoly/Yandex.Disk.localized/my%20projects/karaoke/karaoke-lrc-maker-mvp/src/utils/sharing.ts) временно не задействованы в пользовательском интерфейсе.*

## ⌨️ Горячие клавиши (Hotkeys)

Работают глобально в режиме записи таймингов (когда фокус ввода не находится в текстовом поле):

| Клавиша | Действие |
| :--- | :--- |
| **Space** | Установить тайминг для текущей строки / слова / слога |
| **Backspace** | Отменить последний установленный тайминг фраз |
| **← / →** | Перемотка аудиоплеера на 2 секунды назад / вперед |
| **P** | Начать воспроизведение / поставить на паузу |

## 🚀 Запуск и разработка

Для локального запуска и сборки проекта потребуется [Node.js](https://nodejs.org/).

1. **Установка зависимостей:**
   ```bash
   npm install
   ```

2. **Запуск сервера разработки (Vite 8):**
   ```bash
   npm run dev
   ```

3. **Сборка production-версии (с плагином inline singlefile):**
   ```bash
   npm run build
   ```

4. **Предпросмотр сборки:**
   ```bash
   npm run preview
   ```
